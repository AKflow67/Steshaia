# STESHAÏA Bijoux - Site vitrine

Site vitrine statique (HTML/CSS/JS, un seul `index.html`) avec boutique **Ecwid** embarquée.
Réalisé par **WebKlein - L'Artisan du Web** (https://webklein.fr).

## Contenu
- `index.html` - page unique (CSS + JS inline)
- `hero-unboxing.mp4` + `hero-unboxing-poster.jpg` - vidéo hero (muette, en boucle) + image-poster
- `imageslogo.webp` / `imageslogo-complet.webp` - logos optimisés
- `ecwid-theme.css` - thème CSS à coller dans l'admin Ecwid (Design) quand la boutique est branchée
- `contact.php` - formulaire de contact (fonctionne sur hébergement PHP type OVH ; inactif sur Netlify)

## Hébergement
- Preview / staging : **Netlify**
- Hébergement final prévu : **OVH Starter** (domaine + mail + SSL)

## Boutique Ecwid
Bloc d'intégration commenté dans `index.html` (section Boutique). Remplacer `STORE_ID` par l'ID de la boutique Stesha, décommenter, retirer la grille de démonstration.
